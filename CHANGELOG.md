# Changelog

All notable changes to the LessOTP SDKs.

## 0.1.0 — 2026-06-20

Initial release.

- PHP — `lessotp/sdk`
- JavaScript / Bun / Node — `@lessotp/sdk`
- Go — `github.com/lessotp/sdk/go`
- Python — `lessotp-sdk`

Surface: `authRequest` / `auth_request`, constructor-level `environment`, optional per-call environment override, `verifyWebhookSignature`, and `parseVerifiedWebhook`.
